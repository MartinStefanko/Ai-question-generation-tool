## Spustenie projektu

1. Vytvorte virtuálne prostredie:
   python -m venv venv

2. Aktivujte ho:
   venv\Scripts\Activate
   # source venv/bin/activate  # Linux / macOS

3. Nainštalujte závislosti:
   pip install -r requirements.txt

4. Spustite aplikáciu:
   streamlit run Hlavna_cast/app.py
